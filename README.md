# 🎯 Number Guessing Game

## 📖 Description

The Number Guessing Game is a simple interactive application where the user tries to guess a randomly generated number. The system provides hints whether the guessed number is too high or too low until the correct number is found.

## 🚀 Features

* Random number generation
* User input for guessing
* Hints (Too High / Too Low)
* Tracks number of attempts
* Simple and interactive gameplay
* GUI support (if using Tkinter)

## 🛠 Technologies Used

* Python
* Tkinter (for GUI, if used)

## 🎯 How It Works

1. The system generates a random number.
2. The user enters a guess.
3. The program compares the guess with the actual number.
4. It gives hints:

   * Too High
   * Too Low
5. The process continues until the correct number is guessed.
